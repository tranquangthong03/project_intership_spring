# Data Collection Policy

## Hard rules

- only use publicly accessible pages that are allowed to be accessed
- do not bypass login walls
- do not bypass captcha
- do not bypass rate limits
- do not bypass anti-bot protections
- do not scrape private groups or private communities
- do not implement unsafe Facebook group scraping
- do not implement risky source access

## Safe development preference

For early milestones, prefer:

- fixture-based data
- manual JSON imports
- source adapter skeletons
- normalization using sample HTML or saved examples
